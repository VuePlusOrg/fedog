# fedog

## Introduction
An excellent front-end monitoring platform.

## Project Instructions
### Dependency Installation
```
pnpm install
```

### Service Run
```
pnpm serve
```

### Service Build
```
pnpm build
```

### Code Lint
#### Lint Fixes
```
pnpm lint
```

#### Lint Check
```
pnpm lint:check
```

### Code Commit
```
pnpm commit
```
